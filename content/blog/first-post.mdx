# Website Revamping

## The history
Yesterday, I decided to rewrite my portfolio since the previous one was pretty simple and not **unique**. It was kinda dropped since I mostly focus on personal projects.
I was kinda tired so I decided to do it the next day. Today, I started writing the frontend using [**NextJS**](https://nextjs.org/), [**shadcn**](https://ui.shadcn.com/) & [**magic-ui**](https://magicui.design/), which are amazing frontend-developing tools & ui kits.

## How it was
Rewriting was such a simple task, I mostly took some designs from olds projects, who were unfinished. I also used some parts of the old portfolio, which was still good.
To write that new portfolio, i've spent over **5 hours** re-designing new components & making the blog using [**Velite**](https://velite.js.org/)
